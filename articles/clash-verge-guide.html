新一代客户端 Clash Verge 配置与使用教程

📅 发布日期：2026-06-03 🏷️ 分类：客户端 👤 作者：椰汁评测组

在 2023 年底，大家熟知的 Clash for Windows (CFW) 以及 ClashX 开发者相继宣布停止维护，让许多网络加速用户一时间手足无措。幸运的是，开源社区迅速推出了接棒的替代者——Clash Verge Rev。它使用 Rust 语言配合极速的 Tauri 框架重构，不仅界面极其美观現代，而且内核无缝支持最新的 Mihomo (Clash Meta) 内核，支持更复杂的加密协议。本篇文章将教大家如何快速上手配置这款主流客户端。

一、安装与首选中文汉化切换

首先，请在我们的 小白教程 页面下载适用于您系统平台的最新安装包（注意区分 Windows x64 与 macOS 的不同版本芯片后缀）。

安装完成后，软件默认展示为英文界面。我们可以通过以下简单的两步切换为中文：
1. 打开软件，点击左侧菜单栏底部的 【Settings (设置)】。
2. 找到 【Language】 下拉菜单，点击选中 【中文 (Chinese)】。无需重启，整个软件界面就会瞬间变为熟悉的中文环境。

二、订阅配置文件一键导入

汉化完成后，我们就可以导入机场的订阅节点了：

  1. 进入您的机场控制台，点击【复制 Clash 订阅链接】。
  2. 打开 Clash Verge，点击左侧菜单的【订阅】。
  3. 在顶部输入框中粘贴刚才复制的订阅网址链接，点击右侧的【导入 (Import)】按钮。
  4. 此时，订阅下方会刷出机场的配置文件卡片，鼠标左键单击选中该卡片。当卡片边缘亮起蓝色描边时，即代表该配置文件已成功在本地生效。

三、激活代理与路由分流模式选择

配置文件导入后，我们需要开启系统接管代理：
点击左侧的【设置】 -> 找到【系统代理 (System Proxy)】选项 -> 将右侧的开关点击**点亮**。此时，你的系统流量就已经开始经过 Clash 进行流转了。

为了获得最流畅的体验，点击左侧菜单的【代理】，在页面最顶部选择【规则 (Rule)】模式。此时访问百度、网银依然直连，访问 GitHub、Google 则走专线网络。

你可以在代理组（通常是 PROXY 或 节点选择）中，点开下拉列表选择具体的地区节点。点击节点右侧闪电图标可以测试其真实延迟，优先选用延迟低（绿色）的服务器节点。

四、进阶配置：激活 TUN 内核虚拟网卡

如果你发现某些客户端（如 Steam 联机游戏、模拟器、Git 命令端）无法正常走加速，这是因为系统代理没有覆盖非浏览器流量。我们需要开启 TUN 模式：
首先，退出软件。在桌面上右键点击 Clash Verge 图标，选择【以管理员身份运行】。进入【设置】-> 找到【TUN 模式 (TUN Mode)】并开启。此时，软件会接管你网卡的全部收发流量，各种大型游戏或终端软件就能获得完美的出海加速能力了。
