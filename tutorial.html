网络加速客户端新手配置教程

本教程汇总了主流系统平台下开源网络连接客户端的下载、配置订阅与模式选择，帮助您快速熟悉使用方法。

01

获取订阅链接

在您购买的机场后台（如飞猫云、唯兔云等），找到【快速导入】或【便捷订阅】区域，点击【复制 Clash 订阅链接】。请务必妥善保管该链接，切勿泄露给他人。

02

下载安装 Clash Verge

Windows 端推荐使用新一代图形化客户端 Clash Verge Rev。它支持中文，界面极简，底层使用的是 Meta 高性能内核，稳定抗封锁。

Clash Verge Rev (x64) Windows 推荐

开源安全的全新 Clash 客户端。支持规则分流、TUN 虚拟网卡模式，自带中文包。

GitHub 官方下载 备用高速下载
03

导入配置文件

打开 Clash Verge 客户端，点击左侧菜单栏的【订阅 (Profiles)】。在顶部的输入框中粘贴刚才在第一步复制的【Clash订阅链接】，然后点击右侧的【导入 (Import)】按钮。导入成功后，选中该配置文件使其亮起激活。

04

开启代理，选择模式

点击左侧菜单栏的【设置 (Settings)】，将【系统代理 (System Proxy)】开关打开。然后点击左侧的【代理 (Proxies)】，在上方选择模式：
• 规则模式 (Rule，推荐)：自动分流，国内网站直连以节省机场流量，国外网站走加速。
• 全局模式 (Global)：所有流量都走机场加速（国内网页会变慢）。
• 直连模式 (Direct)：不使用任何加速节点。

安全提醒： 强烈建议在代理设置中选用“规则模式”。使用全局模式可能会导致国内网银、微信等无法正常登录或提示IP异常。
01

复制机场订阅链接

登录您的机场账户，进入用户中心，在订阅区域点击【复制 Clash 订阅】。不要在公共场合分享此专属链接。

02

下载 Clash Verge macOS 版

对于苹果电脑用户，我们同样推荐使用 Clash Verge Rev macOS版。若您的 Mac 是 M1/M2/M3 系列芯片，请选择 aarch64 版本；如果是旧款 Intel 芯片，请选择 x64 版本。

Clash Verge Mac (aarch64 / x64) macOS 推荐

支持 M 系列及 Intel 芯片 Mac。对 macOS 系统的功耗与内存占用进行了专门优化。

GitHub 官方下载 官方安装包
03

导入并启用配置

打开 Clash Verge。前往【Profiles】界面，在顶部输入框粘入订阅链接，点击【Import】。成功加载后，点击选中新导入的配置文件名称。

04

启动并选择节点

进入 macOS 左侧的 【Settings】，将 【System Proxy】 设置为开启。在 【Proxies】 面板中，选择 【Rule (规则模式)】 并在节点组里点选任意香港或日本节点（绿色延迟数越小速度越快）。

01

复制 Clash 订阅地址

登录机场控制台，找到【一键导入/订阅复制】，点击【复制 Clash 订阅地址】，然后将其发送或在手机上粘贴保存。

02

下载 Clash for Android

安卓平台首选 Clash for Android 客户端（简称 CFA）。其操作界面高度原生化，资源占用极低，且运行极其稳定。

Clash for Android Android 推荐

安卓端最成熟的加速网络客户端。支持全局/绕过大陆等分流策略。

GitHub 官方发布页 APK 安装包下载
03

导入订阅文件

打开 Clash for Android 软件，点击【配置 (Profiles)】 -> 【新配置 (New Profile)】 -> 【从URL导入 (URL)】。在“URL”一栏中粘贴第一步复制的订阅链接，并在“名称”栏输入机场名字，点击右上角软盘图标保存。最后勾选选中该配置。

04

点亮连接开关

返回客户端主页，点击【已停止 (Stopped)】灰色大按钮，在弹出的 VPN 连接请求中点击【确定】，按钮变成橙色【已运行】即代表连接成功。随后可在【控制面板】里的“代理”中点选具体节点。

01

下载 Shadowrocket (小火箭)

在 iOS (iPhone / iPad) 上，网络加速最推荐使用 Shadowrocket（俗称小火箭）。
⚠️ 注意： 苹果 App Store 国区无法下载该软件。您需要使用港区或美区 Apple ID 登录 App Store 购买下载（售价约 $2.99 ）。您可以向购买的机场客服或相关渠道租用/购买海外 Apple ID。

02

获取链接一键导入

用 iPhone 浏览器登录机场后台，进入控制面板，在【一键导入】区域点击【一键导入 Shadowrocket】，手机会自动唤起小火箭并自动配置。
如果没有一键导入选项，请点击【复制小火箭/SS订阅链接】，然后打开小火箭，软件会自动弹窗检测并询问是否添加该订阅链接，点击“确定”即可。

03

测试节点延迟

导入完成后，订阅列表下会出现一长串国家和地区节点。点击【连通性测试 (Latency Test)】，每个节点右侧会刷出具体的 Ping 值，建议选择 100ms 以下的节点使用。

04

选择模式并启动连接

将底部的【全局路由 (Global Routing)】设置为【配置 (Config，相当于规则模式)】。最后点击顶部的【未连接 (Not Connected)】开关，在弹出的系统 VPN 权限申请中输入 iPhone 锁屏密码即可开始使用。

常见问题深度问答 FAQs

RESOLVING COMMON PITFALLS
1. 连上客户端后，为什么微信、淘宝等国内软件很慢或者完全打不开？
这是因为您错误地开启了【全局模式 (Global)】。在全局模式下，您访问国内网站的请求也会发送至机场节点，再由节点绕回国内，这会导致延迟急剧增加。
解决方案： 请将客户端的路由模式切换为【规则模式 (Rule)】或【配置 (Config)】，分流规则会自动识别国内域名和IP，使其不需要绕行国外节点，保证国内软件满速运行。
2. 导入订阅链接时提示 "Invalid Config"、"500 Internal Error" 或解析失败？
这通常由以下三个原因导致：
① 订阅链接未被正确激活：部分机场购买套餐后可能需要等待数分钟以配置服务器节点；或者您的套餐已经过期或流量超支。
② 复制了错误的链接格式：比如将 V2ray 的订阅格式强行导入到 Clash 中。请确认你在机场后台点击的是“Clash 订阅”字样的复制按钮。
③ 网络拥堵或被运营商阻断：您可以尝试暂时断开本地 WiFi 切换为手机热点，或者开启已有的其他代理再行下载更新订阅。
3. 节点后面的 1x、2x、0.1x 究竟是什么含义？我该怎么选？
这是机场的流量计费倍率。它决定了你实际消耗的套餐流量：
• 0.1x 倍率：你用了 10GB 流量，机场只扣除 1GB 的额度，适合重度下载或看长视频。
• 1.0x 倍率：标准计费，用多少扣多少，为大部分常规节点。
• 2.0x / 5.0x 倍率：通常是顶级 IEPL 游戏专线节点，由于专线带宽极其昂贵，使用该节点消耗的流量额度会翻倍。
4. 什么是 TUN 模式？我们在什么情况下需要开启它？
传统的“系统代理”仅对支持系统代理解析的浏览器和聊天工具起效。而TUN 模式会在系统底层安装一个虚拟网卡，将整个操作系统所有的网络流量（包含命令终端、某些特定主机游戏、UWP应用如微软商店等）强行拉入 Clash 的解析规则中。
如果你需要进行终端开发（Git / Docker 拉取镜像），或者需要游玩某些必须要外网登录的主机和单机游戏，建议开启客户端中的 【TUN 模式 (TUN Mode)】。
